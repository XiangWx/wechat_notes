live-server 快速搭建临时服务器：
1.npm install -g live-server
2.项目初始化 npm init （一路回车）
3.在package.json中增加start中增加新的脚本
"scripts": {
  "server": "live-server ./ --port=9090"
}
4.live-server 笔记跑起来
